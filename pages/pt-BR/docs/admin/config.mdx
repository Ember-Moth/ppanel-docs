## Variáveis de Ambiente

Antes de implantar, certifique-se de definir diretamente as seguintes variáveis de ambiente no comando de implantação para configurar o aplicativo:

```bash
# Idioma padrão do aplicativo
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Seu idioma padrão zh-CN | en-US>

# URL do site e ponto de extremidade da API
NEXT_PUBLIC_SITE_URL=<URL do seu site>
NEXT_PUBLIC_API_URL=<URL da sua API>

# Credenciais padrão do usuário (para configuração inicial ou fins de teste)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Seu e-mail de usuário padrão>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Sua senha de usuário padrão>
```

Você pode consultar um [arquivo de ambiente de exemplo](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) para ver um arquivo de referência estruturado.

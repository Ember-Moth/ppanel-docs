## คำอธิบายศัพท์

คำศัพท์บางคำใน PPanel อาจแตกต่างจากระบบแผงควบคุมอื่น ๆ เพื่อให้แน่ใจว่าคุณสามารถเข้าใจเนื้อหาของเอกสารได้อย่างถูกต้องและหลีกเลี่ยงความเข้าใจผิด ขอแนะนำให้คุณทำความเข้าใจคำศัพท์ต่อไปนี้ก่อนอ่าน:

- **ส่วนผู้ใช้**
  เป็นส่วนติดต่อที่ให้บริการแก่ผู้ใช้สุดท้าย ผู้ใช้สามารถโต้ตอบกับระบบผ่านส่วนนี้ คุณสามารถปรับแต่งหรือสร้างใหม่ตามความต้องการเพื่อให้เหมาะสมกับการปรับแต่งเว็บไซต์ของคุณ

- **ส่วนผู้ดูแลระบบ**
  เป็นส่วนติดต่อที่ใช้สำหรับการดำเนินการของผู้ดูแลระบบ รับผิดชอบในการจัดการระบบ ผู้ใช้ และข้อมูล คุณสามารถปรับแต่งหรือสร้างใหม่ตามความต้องการเพื่อให้เหมาะสมกับความต้องการในการจัดการของคุณ

- **ส่วนบริการ**
  เป็นชั้น API ของ PPanel ที่จัดการการแลกเปลี่ยนข้อมูลทั้งหมดกับส่วนหน้า รับผิดชอบในการดำเนินธุรกิจและการให้บริการข้อมูล

- **ส่วนโหนด**
  รับผิดชอบการสื่อสารระหว่างส่วนบริการของ PPanel กับโหนดต่าง ๆ (ปลายทาง) เพื่อให้แน่ใจว่าการเชื่อมต่อของโหนดในเครือข่ายและความเสถียรของบริการ

- **ส่วนลูกค้า**
  เป็นโปรแกรมที่ผู้ใช้ใช้เชื่อมต่อกับระบบ โดยทั่วไปหมายถึงซอฟต์แวร์หรือแอปพลิเคชันที่อยู่บนอุปกรณ์ของผู้ใช้ รับผิดชอบในการสร้างการเชื่อมต่อกับระบบและใช้บริการที่เกี่ยวข้อง

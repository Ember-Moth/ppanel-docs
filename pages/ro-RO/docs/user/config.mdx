## Variabile de mediu

Înainte de desfășurarea aplicației, asigurați-vă că setați direct următoarele variabile de mediu în comanda de desfășurare pentru a configura aplicația:

```bash
# Limba implicită a aplicației
NEXT_PUBLIC_DEFAULT_LANGUAGE=<limba dumneavoastră implicită ro-RO | en-US>

# URL-ul site-ului și endpoint-urile API
NEXT_PUBLIC_SITE_URL=<URL-ul dumneavoastră pentru site>
NEXT_PUBLIC_API_URL=<URL-ul dumneavoastră pentru API>

# Email de contact pentru suport sau consultanță
NEXT_PUBLIC_EMAIL=<email-ul dumneavoastră de contact>

# Linkuri pentru comunitate și rețele sociale
NEXT_PUBLIC_TELEGRAM_LINK=<linkul dumneavoastră Telegram>
NEXT_PUBLIC_TWITTER_LINK=<linkul dumneavoastră Twitter>
NEXT_PUBLIC_DISCORD_LINK=<linkul dumneavoastră Discord>
NEXT_PUBLIC_INSTAGRAM_LINK=<linkul dumneavoastră Instagram>
NEXT_PUBLIC_LINKEDIN_LINK=<linkul dumneavoastră LinkedIn>
NEXT_PUBLIC_FACEBOOK_LINK=<linkul dumneavoastră Facebook>
NEXT_PUBLIC_GITHUB_LINK=<linkul dumneavoastră GitHub>

# Credențiale implicite pentru utilizatori (pentru configurări inițiale sau scopuri de testare)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<email-ul dumneavoastră implicit pentru utilizator>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<parola dumneavoastră implicită pentru utilizator>

# Home Page Settings
NEXT_PUBLIC_HOME_USER_COUNT=999
NEXT_PUBLIC_HOME_SERVER_COUNT=999
NEXT_PUBLIC_HOME_LOCATION_COUNT=999
```

Puteți consulta [fișierul de mediu exemplu](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) pentru a vedea un fișier de referință structurat.

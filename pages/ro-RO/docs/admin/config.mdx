## Variabile de mediu

Înainte de desfășurare, asigurați-vă că setați direct următoarele variabile de mediu în comanda de desfășurare pentru a configura aplicația:

```bash
# Limba implicită a aplicației
NEXT_PUBLIC_DEFAULT_LANGUAGE=<limba dumneavoastră implicită zh-CN | en-US>

# URL-ul site-ului și endpoint-urile API
NEXT_PUBLIC_SITE_URL=<URL-ul dumneavoastră pentru site>
NEXT_PUBLIC_API_URL=<URL-ul dumneavoastră pentru API>

# Credențiale implicite pentru utilizatori (pentru configurare inițială sau scopuri de testare)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<email-ul dumneavoastră implicit pentru utilizator>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<parola dumneavoastră implicită pentru utilizator>
```

Puteți consulta [fișierul de mediu exemplu](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) pentru a vedea un fișier de referință structurat.

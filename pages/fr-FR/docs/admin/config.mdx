## Variables d'environnement

Avant le déploiement, veuillez vous assurer de définir directement les variables d'environnement suivantes dans la commande de déploiement pour configurer l'application :

```bash
# Langue par défaut de l'application
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Votre langue par défaut zh-CN | en-US>

# URL du site et point de terminaison API
NEXT_PUBLIC_SITE_URL=<L'URL de votre site>
NEXT_PUBLIC_API_URL=<L'URL de votre API>

# Adresse e-mail pour le support ou les consultations
NEXT_PUBLIC_EMAIL=<Votre adresse e-mail de contact>

# Liens vers la communauté et les réseaux sociaux
NEXT_PUBLIC_TELEGRAM_LINK=<Votre lien Telegram>
NEXT_PUBLIC_TWITTER_LINK=<Votre lien Twitter>
NEXT_PUBLIC_DISCORD_LINK=<Votre lien Discord>
NEXT_PUBLIC_INSTAGRAM_LINK=<Votre lien Instagram>
NEXT_PUBLIC_LINKEDIN_LINK=<Votre lien LinkedIn>
NEXT_PUBLIC_FACEBOOK_LINK=<Votre lien Facebook>
NEXT_PUBLIC_GITHUB_LINK=<Votre lien GitHub>

# Identifiants utilisateur par défaut (pour la configuration initiale ou des tests)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Votre adresse e-mail utilisateur par défaut>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Votre mot de passe utilisateur par défaut>
```

Vous pouvez consulter un [exemple de fichier d'environnement](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) pour voir un fichier de référence structuré.

## Déploiement en un clic

[![Déployé sur Zeabur](https://zeabur.com/deployed-on-zeabur-dark.svg)](https://zeabur.com?referralCode=PPanelGroup&utm_source=PPanelGroup&utm_campaign=oss)

Panel est un outil de panneau proxy open source pur, professionnel et parfait, conçu pour être votre choix idéal pour l'apprentissage et l'utilisation pratique.

## Comment utiliser

- Vous pouvez accéder à votre serveur ppanel pour la configuration initiale en visitant `nomdudomaine.zeabur.app`.
- Vous pouvez accéder à votre interface de gestion ppanel pour les opérations de gestion en visitant `nomdudomaineadmin.zeabur.app`.
- Vous pouvez accéder à votre interface utilisateur ppanel pour les opérations d'utilisation en visitant `nomdudomaineutilisateur.zeabur.app`.

## Remarques

- Après avoir mis à jour les noms de domaine du serveur, de l'interface de gestion et de l'interface utilisateur, veuillez mettre à jour les variables d'environnement correspondantes et redémarrer le service.

## Autres

- Pour la configuration des variables d'environnement, veuillez consulter la [documentation officielle de PPanel](https://ppanel.dev/)

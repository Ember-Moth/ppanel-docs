## Однокнопкове розгортання

[![Розгорнуто на Zeabur](https://zeabur.com/deployed-on-zeabur-dark.svg)](https://zeabur.com?referralCode=PPanelGroup&utm_source=PPanelGroup&utm_campaign=oss)

Panel — це чистий, професійний та досконалий інструмент для проксі-панелей з відкритим кодом, який має стати ідеальним вибором для вашого навчання та практичного використання.

## Як користуватися

- Ви можете отримати доступ до вашого ppanel сервера для початкової конфігурації, перейшовши за адресою `серверне_доменне_ім'я.zeabur.app`.
- Ви можете отримати доступ до вашого ppanel адміністративного інтерфейсу для управлінських операцій, перейшовши за адресою `адміністративне_доменне_ім'я.zeabur.app`.
- Ви можете отримати доступ до вашого ppanel користувацького інтерфейсу для використання, перейшовши за адресою `користувацьке_доменне_ім'я.zeabur.app`.

## Зверніть увагу

- Після зміни серверного, адміністративного та користувацького доменів, будь ласка, своєчасно оновіть відповідні змінні середовища та перезапустіть сервіс.

## Інше

- Для налаштування змінних середовища, будь ласка, зверніться до [офіційної документації PPanel](https://ppanel.dev/)

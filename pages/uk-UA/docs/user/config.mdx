## Змінні середовища

Перед розгортанням, будь ласка, переконайтеся, що ви безпосередньо встановили наступні змінні середовища в команді розгортання для налаштування програми:

```bash
# За замовчуванням мова програми
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Ваша мова за замовчуванням uk-UA | en-US>

# URL сайту та API-ендпоінти
NEXT_PUBLIC_SITE_URL=<URL вашого сайту>
NEXT_PUBLIC_API_URL=<Ваш API URL>

# Контактний email для підтримки або консультацій
NEXT_PUBLIC_EMAIL=<Ваш контактний email>

# Посилання на спільноти та соціальні мережі
NEXT_PUBLIC_TELEGRAM_LINK=<Ваше посилання на Telegram>
NEXT_PUBLIC_TWITTER_LINK=<Ваше посилання на Twitter>
NEXT_PUBLIC_DISCORD_LINK=<Ваше посилання на Discord>
NEXT_PUBLIC_INSTAGRAM_LINK=<Ваше посилання на Instagram>
NEXT_PUBLIC_LINKEDIN_LINK=<Ваше посилання на LinkedIn>
NEXT_PUBLIC_FACEBOOK_LINK=<Ваше посилання на Facebook>
NEXT_PUBLIC_GITHUB_LINK=<Ваше посилання на GitHub>

# Облікові дані користувача за замовчуванням (для початкового налаштування або тестування)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Ваш email користувача за замовчуванням>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Ваш пароль користувача за замовчуванням>

# Home Page Settings
NEXT_PUBLIC_HOME_USER_COUNT=999
NEXT_PUBLIC_HOME_SERVER_COUNT=999
NEXT_PUBLIC_HOME_LOCATION_COUNT=999
```

Ви можете ознайомитися з [прикладом файлу середовища](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) для перегляду структурованого зразка файлу.

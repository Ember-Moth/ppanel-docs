## Ortam Değişkenleri

Dağıtım öncesinde, uygulamayı yapılandırmak için aşağıdaki ortam değişkenlerini doğrudan dağıtım komutunda ayarladığınızdan emin olun:

```bash
# Uygulamanın varsayılan dili
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Varsayılan diliniz tr-TR | en-US>

# Site URL'si ve API uç noktası
NEXT_PUBLIC_SITE_URL=<Site URL'niz>
NEXT_PUBLIC_API_URL=<API URL'niz>

# Destek veya danışma için iletişim e-posta adresi
NEXT_PUBLIC_EMAIL=<İletişim e-posta adresiniz>

# Topluluk ve sosyal medya bağlantıları
NEXT_PUBLIC_TELEGRAM_LINK=<Telegram bağlantınız>
NEXT_PUBLIC_TWITTER_LINK=<Twitter bağlantınız>
NEXT_PUBLIC_DISCORD_LINK=<Discord bağlantınız>
NEXT_PUBLIC_INSTAGRAM_LINK=<Instagram bağlantınız>
NEXT_PUBLIC_LINKEDIN_LINK=<LinkedIn bağlantınız>
NEXT_PUBLIC_FACEBOOK_LINK=<Facebook bağlantınız>
NEXT_PUBLIC_GITHUB_LINK=<GitHub bağlantınız>

# Varsayılan kullanıcı kimlik bilgileri (ilk ayar veya test amaçları için)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Varsayılan kullanıcı e-posta adresiniz>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Varsayılan kullanıcı şifreniz>
```

Yapılandırılmış bir referans dosyası görmek için [örnek ortam dosyasına](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) göz atabilirsiniz.

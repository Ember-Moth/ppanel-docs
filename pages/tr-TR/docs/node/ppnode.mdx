# PPanel-node

Çeşitli çekirdekler üzerine inşa edilmiş bir PPanel düğüm sunucusu, V2bX'ten değiştirilmiştir ve V2ay, Trojan, Shadowsocks, Tuic, Hysteria protokollerini destekler.

### Tek Tıkla Kurulum

```
wget -N https://raw.githubusercontent.com/wyx2685/ppanel-node/master/Scripts/install.sh && bash install.sh
```

## Derleme

```bash
# Derlenecek çekirdekleri -tags seçeneği ile belirtin, isteğe bağlı xray, sing
go build -v -o ./node -tags "xray sing with_reality_server with_quic with_grpc with_utls with_wireguard with_acme" -trimpath -ldflags "-s -w -buildid="
```

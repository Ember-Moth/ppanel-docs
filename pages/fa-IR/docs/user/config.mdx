## متغیرهای محیطی

قبل از استقرار، لطفاً اطمینان حاصل کنید که متغیرهای محیطی زیر را به طور مستقیم در دستور استقرار تنظیم کنید تا برنامه پیکربندی شود:

```bash
# زبان پیش‌فرض برنامه
NEXT_PUBLIC_DEFAULT_LANGUAGE=<زبان پیش‌فرض شما fa-IR | en-US>

# URL سایت و نقطه پایانی API
NEXT_PUBLIC_SITE_URL=<URL سایت شما>
NEXT_PUBLIC_API_URL=<URL API شما>

# ایمیل تماس برای پشتیبانی یا مشاوره
NEXT_PUBLIC_EMAIL=<ایمیل تماس شما>

# لینک‌های جامعه و رسانه‌های اجتماعی
NEXT_PUBLIC_TELEGRAM_LINK=<لینک تلگرام شما>
NEXT_PUBLIC_TWITTER_LINK=<لینک توییتر شما>
NEXT_PUBLIC_DISCORD_LINK=<لینک دیسکورد شما>
NEXT_PUBLIC_INSTAGRAM_LINK=<لینک اینستاگرام شما>
NEXT_PUBLIC_LINKEDIN_LINK=<لینک لینکدین شما>
NEXT_PUBLIC_FACEBOOK_LINK=<لینک فیسبوک شما>
NEXT_PUBLIC_GITHUB_LINK=<لینک گیت‌هاب شما>

# اعتبارنامه‌های پیش‌فرض کاربر (برای تنظیمات اولیه یا اهداف آزمایشی)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<ایمیل کاربر پیش‌فرض شما>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<رمز عبور کاربر پیش‌فرض شما>
```

می‌توانید به [فایل نمونه محیطی](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) مراجعه کنید تا ساختار فایل مرجع را مشاهده کنید.

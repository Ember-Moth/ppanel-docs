## Переменные окружения

Перед развертыванием убедитесь, что вы напрямую установили следующие переменные окружения в команде развертывания для настройки приложения:

```bash
# Язык по умолчанию для приложения
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Ваш язык по умолчанию zh-CN | en-US>

# URL сайта и конечная точка API
NEXT_PUBLIC_SITE_URL=<Ваш URL сайта>
NEXT_PUBLIC_API_URL=<Ваш URL API>

# Учетные данные пользователя по умолчанию (для первоначальной настройки или тестирования)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Ваш адрес электронной почты пользователя по умолчанию>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Ваш пароль пользователя по умолчанию>
```

Вы можете обратиться к [примеру файла окружения](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) для просмотра структурированного образца файла.

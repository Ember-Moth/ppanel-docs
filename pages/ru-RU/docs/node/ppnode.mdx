# PPanel-node

Сервер PPanel на основе различных ядер, модифицированный от V2bX, поддерживает протоколы V2ay, Trojan, Shadowsocks, Tuic, Hysteria.

### Установка одним нажатием

```
wget -N https://raw.githubusercontent.com/wyx2685/ppanel-node/master/Scripts/install.sh && bash install.sh
```

## Сборка

```bash
# Укажите ядро для компиляции с помощью опции -tags, доступные xray, sing
go build -v -o ./node -tags "xray sing with_reality_server with_quic with_grpc with_utls with_wireguard with_acme" -trimpath -ldflags "-s -w -buildid="
```

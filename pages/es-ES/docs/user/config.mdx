## Variables de Entorno

Antes de implementar, asegúrese de establecer directamente las siguientes variables de entorno en el comando de implementación para configurar la aplicación:

```bash
# Idioma predeterminado de la aplicación
NEXT_PUBLIC_DEFAULT_LANGUAGE=<su idioma predeterminado zh-CN | en-US>

# URL del sitio y punto final de la API
NEXT_PUBLIC_SITE_URL=<su URL del sitio>
NEXT_PUBLIC_API_URL=<su URL de la API>

# Correo electrónico de contacto para soporte o consultas
NEXT_PUBLIC_EMAIL=<su correo electrónico de contacto>

# Enlaces a comunidades y redes sociales
NEXT_PUBLIC_TELEGRAM_LINK=<su enlace de Telegram>
NEXT_PUBLIC_TWITTER_LINK=<su enlace de Twitter>
NEXT_PUBLIC_DISCORD_LINK=<su enlace de Discord>
NEXT_PUBLIC_INSTAGRAM_LINK=<su enlace de Instagram>
NEXT_PUBLIC_LINKEDIN_LINK=<su enlace de LinkedIn>
NEXT_PUBLIC_FACEBOOK_LINK=<su enlace de Facebook>
NEXT_PUBLIC_GITHUB_LINK=<su enlace de GitHub>

# Credenciales de usuario predeterminadas (para configuración inicial o propósitos de prueba)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<su correo electrónico de usuario predeterminado>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<su contraseña de usuario predeterminada>

# Home Page Settings
NEXT_PUBLIC_HOME_USER_COUNT=999
NEXT_PUBLIC_HOME_SERVER_COUNT=999
NEXT_PUBLIC_HOME_LOCATION_COUNT=999
```

Puede consultar [el archivo de entorno de ejemplo](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) para ver un archivo de referencia estructurado.

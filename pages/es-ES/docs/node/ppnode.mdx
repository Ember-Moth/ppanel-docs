# PPanel-node

Un servidor de nodo PPanel basado en múltiples núcleos, modificado de V2bX, que soporta los protocolos V2ay, Trojan, Shadowsocks, Tuic y Hysteria.

### Instalación con un solo clic

```
wget -N https://raw.githubusercontent.com/wyx2685/ppanel-node/master/Scripts/install.sh && bash install.sh
```

## Construcción

```bash
# Especifica el núcleo a compilar mediante la opción -tags, opcional xray, sing
go build -v -o ./node -tags "xray sing with_reality_server with_quic with_grpc with_utls with_wireguard with_acme" -trimpath -ldflags "-s -w -buildid="
```

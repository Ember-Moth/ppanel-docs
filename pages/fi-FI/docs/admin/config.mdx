## Ympäristömuuttujat

Ennen käyttöönottoa varmista, että asetat seuraavat ympäristömuuttujat suoraan käyttöönotto-komentoon sovelluksen konfiguroimiseksi:

```bash
# Sovelluksen oletuskieli
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Oletuskielesi zh-CN | en-US>

# Sivuston URL ja API-päätepiste
NEXT_PUBLIC_SITE_URL=<Sivustosi URL>
NEXT_PUBLIC_API_URL=<API URL:si>

# Oletuskäyttäjän tunnistetiedot (alkuasetuksia tai testitarkoituksia varten)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Oletuskäyttäjän sähköpostisi>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Oletuskäyttäjän salasanasi>
```

Voit tarkastella [esimerkkitiedostoa](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) rakenteellisen viite-tiedoston katsomiseksi.

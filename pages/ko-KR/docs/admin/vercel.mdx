아래 버튼을 클릭하여 빠르게 배포하세요:

**단계:**

1. [Vercel](https://vercel.com/)에 로그인하거나 등록합니다.
2. 배포 버튼을 클릭하여 저장소를 포크하고 `apps/admin`을 선택합니다.
3. 환경 변수를 구성합니다.
4. **배포**를 클릭하여 배포를 시작합니다.

## 환경 변수

배포 전에 다음 환경 변수를 배포 명령에 직접 설정하여 애플리케이션을 구성해야 합니다:

```bash
# 애플리케이션의 기본 언어
NEXT_PUBLIC_DEFAULT_LANGUAGE=<당신의 기본 언어 zh-CN | en-US>

# 사이트 URL 및 API 엔드포인트
NEXT_PUBLIC_SITE_URL=<당신의 사이트 URL>
NEXT_PUBLIC_API_URL=<당신의 API URL>

# 지원 또는 상담을 위한 연락 이메일
NEXT_PUBLIC_EMAIL=<당신의 연락 이메일>

# 커뮤니티 및 소셜 미디어 링크
NEXT_PUBLIC_TELEGRAM_LINK=<당신의 Telegram 링크>
NEXT_PUBLIC_TWITTER_LINK=<당신의 Twitter 링크>
NEXT_PUBLIC_DISCORD_LINK=<당신의 Discord 링크>
NEXT_PUBLIC_INSTAGRAM_LINK=<당신의 Instagram 링크>
NEXT_PUBLIC_LINKEDIN_LINK=<당신의 LinkedIn 링크>
NEXT_PUBLIC_FACEBOOK_LINK=<당신의 Facebook 링크>
NEXT_PUBLIC_GITHUB_LINK=<당신의 GitHub 링크>

# 기본 사용자 자격 증명 (초기 설정 또는 테스트 목적)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<당신의 기본 사용자 이메일>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<당신의 기본 사용자 비밀번호>

# Home Page Settings
NEXT_PUBLIC_HOME_USER_COUNT=999
NEXT_PUBLIC_HOME_SERVER_COUNT=999
NEXT_PUBLIC_HOME_LOCATION_COUNT=999
```

구조화된 참조 파일을 보려면 [예제 환경 파일](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template)을 참조하세요.

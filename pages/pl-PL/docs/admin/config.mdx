## Zmienne środowiskowe

Przed wdrożeniem upewnij się, że bezpośrednio w poleceniu wdrożenia ustawiono następujące zmienne środowiskowe w celu skonfigurowania aplikacji:

```bash
# Domyślny język aplikacji
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Twój domyślny język pl-PL | en-US>

# URL witryny i punkt końcowy API
NEXT_PUBLIC_SITE_URL=<Twój URL witryny>
NEXT_PUBLIC_API_URL=<Twój URL API>

# Domyślne dane logowania użytkownika (do początkowej konfiguracji lub celów testowych)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Twój domyślny adres e-mail użytkownika>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Twoje domyślne hasło użytkownika>
```

Możesz odwołać się do [przykładowego pliku środowiskowego](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template), aby zobaczyć zorganizowany plik referencyjny.

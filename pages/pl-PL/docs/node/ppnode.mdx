# PPanel-node

Serwer PPanel oparty na różnych rdzeniach, zmodyfikowany z V2bX, obsługujący protokoły V2ay, Trojan, Shadowsocks, Tuic, Hysteria.

### Instalacja jednym kliknięciem

```
wget -N https://raw.githubusercontent.com/wyx2685/ppanel-node/master/Scripts/install.sh && bash install.sh
```

## Budowanie

```bash
# Użyj opcji -tags, aby określić rdzenie do skompilowania, opcjonalnie xray, sing
go build -v -o ./node -tags "xray sing with_reality_server with_quic with_grpc with_utls with_wireguard with_acme" -trimpath -ldflags "-s -w -buildid="
```

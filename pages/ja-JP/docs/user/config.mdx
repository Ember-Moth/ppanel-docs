## 環境変数

デプロイ前に、デプロイコマンドで以下の環境変数を直接設定してアプリケーションを構成してください：

```bash
# アプリのデフォルト言語
NEXT_PUBLIC_DEFAULT_LANGUAGE=<あなたのデフォルト言語 ja-JP | en-US>

# サイトの URL と API エンドポイント
NEXT_PUBLIC_SITE_URL=<あなたのサイトの URL>
NEXT_PUBLIC_API_URL=<あなたの API URL>

# サポートまたは相談のための連絡先メール
NEXT_PUBLIC_EMAIL=<あなたの連絡先メール>

# コミュニティおよびソーシャルメディアリンク
NEXT_PUBLIC_TELEGRAM_LINK=<あなたの Telegram リンク>
NEXT_PUBLIC_TWITTER_LINK=<あなたの Twitter リンク>
NEXT_PUBLIC_DISCORD_LINK=<あなたの Discord リンク>
NEXT_PUBLIC_INSTAGRAM_LINK=<あなたの Instagram リンク>
NEXT_PUBLIC_LINKEDIN_LINK=<あなたの LinkedIn リンク>
NEXT_PUBLIC_FACEBOOK_LINK=<あなたの Facebook リンク>
NEXT_PUBLIC_GITHUB_LINK=<あなたの GitHub リンク>

# デフォルトのユーザー認証情報（初期設定またはテスト目的）
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<あなたのデフォルトユーザーのメール>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<あなたのデフォルトユーザーのパスワード>

# Home Page Settings
NEXT_PUBLIC_HOME_USER_COUNT=999
NEXT_PUBLIC_HOME_SERVER_COUNT=999
NEXT_PUBLIC_HOME_LOCATION_COUNT=999
```

構造化された参考ファイルについては、[サンプル環境ファイル](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template)を参照してください。

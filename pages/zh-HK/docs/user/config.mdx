## 環境變量

在部署前，請確保在部署命令中直接設置以下環境變量以配置應用程序：

```bash
# 應用的默認語言
NEXT_PUBLIC_DEFAULT_LANGUAGE=<您的默認語言 zh-HK | en-US>

# 網站 URL 和 API 端點
NEXT_PUBLIC_SITE_URL=<您的網站 URL>
NEXT_PUBLIC_API_URL=<您的 API URL>

# 支持或諮詢的聯絡郵箱
NEXT_PUBLIC_EMAIL=<您的聯絡郵箱>

# 社區和社交媒體鏈接
NEXT_PUBLIC_TELEGRAM_LINK=<您的 Telegram 鏈接>
NEXT_PUBLIC_TWITTER_LINK=<您的 Twitter 鏈接>
NEXT_PUBLIC_DISCORD_LINK=<您的 Discord 鏈接>
NEXT_PUBLIC_INSTAGRAM_LINK=<您的 Instagram 鏈接>
NEXT_PUBLIC_LINKEDIN_LINK=<您的 LinkedIn 鏈接>
NEXT_PUBLIC_FACEBOOK_LINK=<您的 Facebook 鏈接>
NEXT_PUBLIC_GITHUB_LINK=<您的 GitHub 鏈接>

# 默認用戶憑證（用於初始設置或測試目的）
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<您的默認用戶郵箱>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<您的默認用戶密碼>
```

可以參考[示例環境文件](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template)來查看結構化的參考文件。

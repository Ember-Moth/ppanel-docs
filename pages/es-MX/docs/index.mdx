## Glosario de Términos

Algunos términos de PPanel difieren de otros sistemas de paneles. Para asegurarse de que pueda comprender con precisión el contenido del documento y evitar malentendidos, se recomienda familiarizarse con los siguientes términos antes de leer:

- **Interfaz de Usuario**
  Es la interfaz proporcionada al usuario final, a través de la cual interactúa con el sistema. Puede personalizar o reestructurar esta interfaz según sus necesidades, logrando una personalización del sitio.

- **Interfaz de Administración**
  Es la interfaz utilizada por los administradores para gestionar el sistema, los usuarios y los datos. Puede personalizar o reestructurar esta interfaz según sus necesidades de gestión.

- **Interfaz de Servicio**
  Es la capa API de PPanel, que maneja todas las interacciones de datos con el frontend, encargándose de la ejecución de la lógica de negocio y la provisión de servicios de datos.

- **Interfaz de Nodo**
  Es responsable de la comunicación entre el servidor de PPanel y los nodos (puntos de acceso), asegurando la conexión de los nodos de red y la estabilidad del servicio.

- **Interfaz de Cliente**
  Es la aplicación que los usuarios utilizan para conectarse al sistema, generalmente se refiere al software o aplicación en el dispositivo del usuario, encargándose de establecer la conexión con el sistema y utilizar los servicios relacionados.

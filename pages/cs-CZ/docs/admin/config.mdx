## Environment Variables

Před nasazením se ujistěte, že v příkazech pro nasazení přímo nastavíte následující proměnné prostředí pro konfiguraci aplikace:

```bash
# Výchozí jazyk aplikace
NEXT_PUBLIC_DEFAULT_LANGUAGE=<váš výchozí jazyk cs-CZ | en-US>

# URL webu a API koncové body
NEXT_PUBLIC_SITE_URL=<URL vašeho webu>
NEXT_PUBLIC_API_URL=<URL vašeho API>

# Výchozí uživatelské přihlašovací údaje (pro počáteční nastavení nebo testovací účely)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<váš výchozí uživatelský e-mail>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<váš výchozí uživatelský heslo>
```

Můžete se podívat na [ukázkový soubor prostředí](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) pro strukturovanou referenci.

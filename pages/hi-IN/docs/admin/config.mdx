## पर्यावरण चर

तैनाती से पहले, कृपया सुनिश्चित करें कि तैनाती आदेश में सीधे निम्नलिखित पर्यावरण चर सेट किए गए हैं ताकि अनुप्रयोग को कॉन्फ़िगर किया जा सके:

```bash
# अनुप्रयोग की डिफ़ॉल्ट भाषा
NEXT_PUBLIC_DEFAULT_LANGUAGE=<आपकी डिफ़ॉल्ट भाषा zh-CN | en-US>

# साइट URL और API अंत बिंदु
NEXT_PUBLIC_SITE_URL=<आपकी साइट URL>
NEXT_PUBLIC_API_URL=<आपका API URL>

# समर्थन या परामर्श के लिए संपर्क ईमेल
NEXT_PUBLIC_EMAIL=<आपका संपर्क ईमेल>

# समुदाय और सोशल मीडिया लिंक
NEXT_PUBLIC_TELEGRAM_LINK=<आपका Telegram लिंक>
NEXT_PUBLIC_TWITTER_LINK=<आपका Twitter लिंक>
NEXT_PUBLIC_DISCORD_LINK=<आपका Discord लिंक>
NEXT_PUBLIC_INSTAGRAM_LINK=<आपका Instagram लिंक>
NEXT_PUBLIC_LINKEDIN_LINK=<आपका LinkedIn लिंक>
NEXT_PUBLIC_FACEBOOK_LINK=<आपका Facebook लिंक>
NEXT_PUBLIC_GITHUB_LINK=<आपका GitHub लिंक>

# डिफ़ॉल्ट उपयोगकर्ता प्रमाणपत्र (प्रारंभिक सेटअप या परीक्षण उद्देश्यों के लिए)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<आपका डिफ़ॉल्ट उपयोगकर्ता ईमेल>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<आपका डिफ़ॉल्ट उपयोगकर्ता पासवर्ड>
```

संरचित संदर्भ फ़ाइल देखने के लिए [नमूना पर्यावरण फ़ाइल](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) को देख सकते हैं।

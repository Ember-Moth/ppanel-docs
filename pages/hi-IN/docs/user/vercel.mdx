## Vercel पर तैनाती

नीचे दिए गए बटन पर क्लिक करके त्वरित तैनाती करें:

[![Vercel पर तैनात करें](https://vercel.com/button)](https://vercel.com/new/clone?demo-description=PPanel%20एक%20शुद्ध%2C%20पेशेवर%2C%20और%20परिपूर्ण%20ओपन-सोर्स%20प्रॉक्सी%20पैनल%20टूल%20है%2C%20जो%20आपके%20सीखने%20और%20व्यावहारिक%20उपयोग%20के%20लिए%20आदर्श%20चुनाव%20के%20रूप%20में%20डिज़ाइन%20किया%20गया%20है&demo-image=https%3A%2F%2Furlscan.io%2Fliveshot%2F%3Fwidth%3D1920%26height%3D1080%26url%3Dhttps%3A%2F%2Fuser.ppanel.dev&demo-title=PPanel%20उपयोगकर्ता%20वेब&demo-url=https%3A%2F%2Fuser.ppanel.dev%2F&from=.&project-name=ppanel-user-web&repository-name=ppanel-web&repository-url=https%3A%2F%2Fgithub.com%2Fperfect-panel%2Fppanel-web&root-directory=apps%2Fuser&skippable-integrations=1)

**चरण:**

1. [Vercel](https://vercel.com/) पर लॉगिन करें या रजिस्टर करें।
2. तैनाती बटन पर क्लिक करें, रिपॉजिटरी को फोर्क करें और `apps/user` चुनें।
3. पर्यावरण चर कॉन्फ़िगर करें।
4. तैनाती शुरू करने के लिए **Deploy** पर क्लिक करें।

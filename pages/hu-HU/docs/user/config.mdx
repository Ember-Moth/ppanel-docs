## Környezeti változók

A telepítés előtt győződjön meg arról, hogy közvetlenül beállította az alábbi környezeti változókat a telepítési parancsban az alkalmazás konfigurálásához:

```bash
# Az alkalmazás alapértelmezett nyelve
NEXT_PUBLIC_DEFAULT_LANGUAGE=<az ön alapértelmezett nyelve hu-HU | en-US>

# Webhely URL és API végpont
NEXT_PUBLIC_SITE_URL=<az ön webhely URL-je>
NEXT_PUBLIC_API_URL=<az ön API URL-je>

# Támogatási vagy konzultációs kapcsolati e-mail
NEXT_PUBLIC_EMAIL=<az ön kapcsolati e-mail címe>

# Közösségi és közösségi média linkek
NEXT_PUBLIC_TELEGRAM_LINK=<az ön Telegram linkje>
NEXT_PUBLIC_TWITTER_LINK=<az ön Twitter linkje>
NEXT_PUBLIC_DISCORD_LINK=<az ön Discord linkje>
NEXT_PUBLIC_INSTAGRAM_LINK=<az ön Instagram linkje>
NEXT_PUBLIC_LINKEDIN_LINK=<az ön LinkedIn linkje>
NEXT_PUBLIC_FACEBOOK_LINK=<az ön Facebook linkje>
NEXT_PUBLIC_GITHUB_LINK=<az ön GitHub linkje>

# Alapértelmezett felhasználói hitelesítő adatok (kezdeti beállításokhoz vagy tesztelési célokra)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<az ön alapértelmezett felhasználói e-mail címe>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<az ön alapértelmezett felhasználói jelszava>

# Home Page Settings
NEXT_PUBLIC_HOME_USER_COUNT=999
NEXT_PUBLIC_HOME_SERVER_COUNT=999
NEXT_PUBLIC_HOME_LOCATION_COUNT=999
```

A struktúrált referencia fájl megtekintéséhez tekintse meg a [példa környezeti fájlt](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template).

## Biến Môi Trường

Trước khi triển khai, hãy đảm bảo rằng bạn đã thiết lập trực tiếp các biến môi trường sau trong lệnh triển khai để cấu hình ứng dụng:

```bash
# Ngôn ngữ mặc định của ứng dụng
NEXT_PUBLIC_DEFAULT_LANGUAGE=<Ngôn ngữ mặc định của bạn zh-CN | en-US>

# URL trang web và điểm cuối API
NEXT_PUBLIC_SITE_URL=<URL trang web của bạn>
NEXT_PUBLIC_API_URL=<URL API của bạn>

# Email liên hệ hỗ trợ hoặc tư vấn
NEXT_PUBLIC_EMAIL=<Email liên hệ của bạn>

# Liên kết cộng đồng và mạng xã hội
NEXT_PUBLIC_TELEGRAM_LINK=<Liên kết Telegram của bạn>
NEXT_PUBLIC_TWITTER_LINK=<Liên kết Twitter của bạn>
NEXT_PUBLIC_DISCORD_LINK=<Liên kết Discord của bạn>
NEXT_PUBLIC_INSTAGRAM_LINK=<Liên kết Instagram của bạn>
NEXT_PUBLIC_LINKEDIN_LINK=<Liên kết LinkedIn của bạn>
NEXT_PUBLIC_FACEBOOK_LINK=<Liên kết Facebook của bạn>
NEXT_PUBLIC_GITHUB_LINK=<Liên kết GitHub của bạn>

# Thông tin xác thực người dùng mặc định (dùng cho thiết lập ban đầu hoặc mục đích thử nghiệm)
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<Email người dùng mặc định của bạn>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<Mật khẩu người dùng mặc định của bạn>
```

Bạn có thể tham khảo [tệp môi trường mẫu](https://github.com/perfect-panel/ppanel-web/blob/main/apps/user/.env.template) để xem tệp tham khảo có cấu trúc.
